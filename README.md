# nauty-sp-demo
Generating graphs with nauty and Traces, and displaying with NetworkX Matplotlib 
Dependencies: nauty and Traces, NetworkX, NetworkX Matplotlib, pip (to install NetworkX)

Versions:
- nauty and Traces: nauty2_9_1
-  NetworkX: 3.6

Done on Windows Machine, Unix Subsystem

nauty and Traces download:
https://pallini.di.uniroma1.it/

NetworkX download:
https://networkx.org/documentation/stable/install.html
